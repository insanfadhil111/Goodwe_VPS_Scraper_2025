README - Scrapping SEMS Portal

1. Persiapan
   - Pastikan Python 3 sudah terinstall
   - Install paket venv:
     sudo apt update
     sudo apt install -y python3 python3-venv python3-pip

2. Buat folder project
   mkdir -p ~/solar
   cd ~/solar

   (copy file scrapping.py ke folder ini)

3. Buat virtual environment
   python3 -m venv venv

4. Aktifkan virtual environment
   source venv/bin/activate

   (untuk keluar: ketik "deactivate")

5. Install dependency
   pip install --upgrade pip
   pip install selenium webdriver-manager requests

   Jika server tanpa GUI:
   sudo apt install -y xvfb
   Jalankan script dengan:
   xvfb-run -a python scrapping.py

6. Menjalankan script
   python scrapping.py

7. Catatan

   - Klik otomatis "Gedung 3 Teknik"
